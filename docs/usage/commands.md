# 使用指南总览

为了让不同角色更容易找到自己真正需要看的内容，使用指南已经拆成 3 份：

- [玩家指南](player-guide.md) — 给普通玩家看，重点是家园、Warp、TPA、登录认证
- [服主指南](owner-guide.md) — 给服主和普通管理员看，重点是安装后如何管理功能、权限、Web 面板
- [技术管理员指南](admin-guide.md) — 给技术管理员看，重点是诊断、修复、共享配置、节点管理、Web API

## 我该看哪一份？

| 你的角色 | 推荐文档 |
|----------|----------|
| 普通玩家 | [玩家指南](player-guide.md) |
| 服主 / 普通管理员 | [服主指南](owner-guide.md) |
| 技术管理员 / 维护人员 | [技术管理员指南](admin-guide.md) |

## 功能总览

CrossServer 当前主要功能如下：

| 分类 | 包含内容 |
|------|----------|
| 玩家功能 | 家园、Warp、TPA、登录认证 |
| 管理功能 | 节点状态、模块开关、共享路由、重载 |
| 诊断功能 | 转服诊断、认证诊断、经济流水、异常修复 |
| 图形菜单 | Homes GUI、Warp GUI、路由管理 GUI、Transfer GUI |
| Web 面板 | 仪表盘、模块、路由、配置文档、节点配置、日志中心、转服诊断 |
| 配置中心 | 共享模块、共享路由、配置文档、节点配置快照 |

## 推荐阅读顺序

1. 玩家先看 [玩家指南](player-guide.md)
2. 服主再看 [服主指南](owner-guide.md)
3. 需要排障时看 [技术管理员指南](admin-guide.md)
